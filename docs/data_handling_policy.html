DATA HANDLING POLICY

WorkScope Collector
データ取扱い方針

Version 1.0 / 2026年5月
対象: [薬局名] 様

本書は、業務記録ツール「WorkScope Collector」が薬局業務PC上で取り扱うデータの収集範囲、マスキング処理、保管期間、削除手順、開示請求窓口および法的根拠を1ページで明示するものです。本書の内容は同意書および運用手順書と整合しています。

1. 収集データ一覧

区分項目処理
収集 アクティブウィンドウのスクリーンショット OCRで個人情報を検出 → 黒塗り後のみ保存。原本は即時破棄
収集 ウィンドウタイトル マスキング辞書で個人情報文字列を黒塗り後に保存
収集 使用アプリ名・操作時刻・滞在時間 JSONLログとしてローカル保存
非収集 キーボード入力内容 キーロガー機能は実装していません
非収集 クリップボード・マイク・カメラ OS API へのアクセス権を要求しません
非収集 パスワード入力画面 入力フィールド検出時にスクリーンショットを破棄

2. マスキング対象一覧

テキスト系

  • 患者氏名（漢字・ひらがな・カタカナ）
  • 保険証番号・記号番号・被保険者番号
  • 生年月日（西暦・和暦・複数形式）
  • 電話番号（固定・携帯・FAX）
  • 住所・郵便番号
  • マイナンバー・処方箋ID

処理保証

  • マスキング処理はすべて端末内で完結
  • マスキング前のオリジナル画像は即時破棄
  • マスキング辞書は薬局ごとにカスタマイズ可
  • 誤検出を発見した場合は即日辞書更新で対応
  • 定期的に薬局担当者による目視サンプリング検証を実施

3. 保管期間

データ種別保管期間保管場所削除方式
イベントログ（操作時刻・アプリ名等） 90日 レセコン端末内 %APPDATA%\WorkScope\data 期限超過分を日次バッチで自動削除
マスク済みスクリーンショット 30日 レセコン端末内 %APPDATA%\WorkScope\data 期限超過分を日次バッチで自動削除
解析側で受領したデータ プロジェクト終了後30日以内 暗号化された解析環境 解析環境ごと安全消去（証明書発行）

4. 削除手順

  1. 個別削除: タスクトレイから「直近スクショを確認」→ 該当画像を選択 → その場で削除可能。
  2. 全データ削除: タスクトレイから「アンインストール」→「データも削除」を選択。%APPDATA%\WorkScope 配下を完全削除。
  3. 解析側からの削除請求: 後述の窓口にご連絡いただくと、解析側に保管中のデータを7営業日以内に削除し、削除証明書を発行します。

5. 開示請求・削除請求の窓口

窓口: [担当者名] / [電話番号] / [メールアドレス]
受付時間: 平日 9:00〜18:00（土日祝・年末年始除く）
対応SLA: 開示・削除請求は受付日から原則7営業日以内に対応。緊急時は当日対応も可能。
本人確認: 同意書のご署名と照合のうえ対応。代理人による請求の場合は委任状を必要とします。

6. 法的根拠

個人情報保護法

  • 第17条: 利用目的の特定（業務棚卸しに限定）
  • 第21条: 取得時の利用目的の通知（同意書による明示）
  • 第27条: 第三者提供の制限（外部送信を行わない）
  • 第33条: 開示請求への対応（7営業日以内）

関連法令

  • 薬剤師法: 守秘義務（第28条）に配慮した自動マスキング設計
  • 医療法・薬局業務指針: 患者情報の漏えい防止措置
  • 労働関連法: 個人の人事評価への利用は行わない（同意書に明記）
  • 特定電子メール法: 本ツールはメール送信機能を実装していないため対象外

7. インシデント発生時の対応

マスキング誤検出・データ漏えいの疑い・端末紛失等のインシデントを覚知した場合、薬局担当者は直ちに上記窓口へご連絡ください。受領後1時間以内に切り分け対応を開始し、影響範囲確定後、本人通知・必要に応じて個人情報保護委員会への報告を実施します。
