INSTALL / 02

インストール手順

USBメモリから3ステップで完了します。所要時間は約5分です。

  1. 1
    USBメモリをレセコンPCに接続
    配布したUSBメモリを薬局PCのUSBポートに接続します。エクスプローラーが自動で開きます。
  2. 2
    install.bat を「管理者として実行」
    USB内の install.bat を右クリック →「管理者として実行」を選択。UAC(ユーザーアカウント制御)が出たら「はい」を選択します。
  3. 3
    完了確認
    画面右下のタスクトレイに緑の丸(録画中アイコン)が出れば完了です。再起動は不要です。

DATA REVIEW / 05

データの確認方法

いつでも「何が記録されているか」を、薬剤師さんご自身が確認できます。

方法1: データフォルダを開く

  • タスクトレイアイコンを右クリック
  • 「データフォルダを開く」を選択
  • 場所: %APPDATA%\WorkScope\data
  • マスク済みPNG・JSONLログが格納

方法2: 直近スクショを確認

  • タスクトレイ右クリック →「直近スクショを確認」
  • 最新10件がサムネイル表示
  • 個人情報が黒塗りされていることを確認可能
  • 気になる画像はその場で削除も可能

もし個人情報が黒塗りされていなかったら

  • 直ちに録画を一時停止し、[連絡先] までご連絡ください
  • 該当画像はその場で削除いただいて構いません
  • マスキング辞書を更新し、即日アップデートで対応します

DATA HANDOVER / 06

データの回収方法

2週間の収集完了後、USBメモリ持参で回収にお伺いします。

  1. 1
    収集終了の通知
    タスクトレイ右クリック →「録画 一時停止」で記録を停止してください。
  2. 2
    データフォルダを丸ごとコピー
    %APPDATA%\WorkScope\data フォルダをUSBメモリにコピーします。所要時間は通常5〜10分です。
  3. 3
    担当者が立ち会い、内容を確認
    マスキングが適切にかかっているかを、薬局担当者と一緒に5サンプルほど目視確認します。
  4. 4
    USBメモリの暗号化と持ち帰り
    BitLocker To Go でUSB全体を暗号化したうえで持ち帰ります。解析完了後、USBは安全消去します。

よくあるご質問

Q. PCの動作が重くなりませんか？
A. CPU使用率は通常2〜4%、メモリは200MB程度です。レセコン業務に支障が出る重さにはなりません。万一重さを感じたら一時停止してご連絡ください。
Q. インターネットに何か送られていませんか？
A. 一切送信しません。ファイアウォールでブロックされても正常動作します。ご不安な場合はネットワーク監視ソフトでご確認ください。
Q. アンインストールすると、これまでのデータも消えますか？
A. 「データも一緒に削除する」「データは残す」の選択肢が出ます。回収前にアンインストールする場合は「データは残す」を選択してください。
