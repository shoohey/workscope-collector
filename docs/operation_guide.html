INSTALL / 02

初日: インストール手順

USBメモリから3ステップで完了します。所要時間は約5分です。

  1. 1
    USBメモリをレセコンPCに接続
    配布したUSBメモリを薬局PCのUSBポートに接続します。エクスプローラーが開いたら、中身（WorkScope.exe / install.bat / uninstall.bat / 同意書PDF）が揃っていることを確認してください。
  2. 2
    install.bat を「管理者として実行」
    USB内の install.bat を右クリック →「管理者として実行」を選択。UAC(ユーザーアカウント制御)が出たら「はい」を選択。黒いコンソール画面に 「[5/5] インストール完了」 と出れば成功です。
  3. 3
    タスクトレイで緑の丸を確認
    画面右下のタスクトレイに緑の丸(録画中アイコン)が出れば完了です。再起動は不要です。続けて次ページの「導入直後チェック」を実施してください。

★ インストール直後の重要事項

  • USB は抜かずにそのまま挿しておいてください（直後チェックで利用）
  • レセコン業務を1〜2件操作してもらう前提です（実患者の操作でOK・マスクされます）
  • 同意書はインストール前に署名済みであることが大前提です

DAY 0 CHECK / 03

導入直後の動作確認チェックリスト

インストール完了後10分以内に、薬局担当者と一緒にすべて確認します。1つでも未達なら導入を見送り、現場で原因確認します。

  • タスクトレイに 緑色のアイコンが表示されている
    タスクトレイの上向き矢印(^)を開いて隠れているアイコンも確認。アイコンを右クリック → ステータスが「録画中」であることを目視。
    ✓ 合格基準: 緑丸 + 「録画中（イベント数: 0以上）」表示
  • レセコンを 1〜2件 操作してもらい、スクショが生成される
    薬剤師さんに通常通りレセコンを操作してもらう（実患者でOK）。1分後にトレイ右クリック →「直近のスクショを確認」で 1 件以上表示されること。
    ✓ 合格基準: 直近スクショ一覧に1件以上表示
  • マスキングが 正しく効いている ことを目視
    直近スクショを実際に開いて、患者氏名・電話・住所・保険者番号・生年月日・マイナンバーが すべて黒塗り されていることを確認。薬剤師さんと一緒に確認するのが理想。
    ✗ 1つでも黒塗りされていない PII を発見したら → 即一時停止 → 担当者に電話
  • PCの動作に明らかな重さが出ていない
    タスクマネージャー(Ctrl+Shift+Esc)で WorkScope.exe の CPU が 5% 以下、メモリが 300MB 以下であること。レセコン操作にラグが出ていないことを薬剤師さんに口頭で確認。
    ✓ 合格基準: CPU 5%以下 / メモリ 300MB以下 / 体感ラグなし
  • 一時停止 → 再開が動作する
    トレイ右クリック →「一時停止」→ アイコンが灰色になる →「再開」→ 緑に戻る、を実機で1往復。薬剤師さんに「いつでも止められる」ことを実演で見せる。
    ✓ 合格基準: アイコンの色変化と「録画中／一時停止中」表示の切替を確認
  • 緊急連絡先カードを薬局担当者に手渡し
    電話番号・LINE ID・メールアドレスをまとめた小カードを、薬局の見える場所に貼ってもらう。「異変があったら即連絡」を口頭で約束。
    ✓ 合格基準: 連絡先が現場に物理的に残っている

TRAY UI / 04

タスクトレイの見方

画面右下のアイコンで、いつでも録画状態を確認・操作できます。色の意味を覚えるだけでOKです。

緑の丸: 録画中業務記録が正常に動作中。これが日常状態です。
グレーの丸: 一時停止中休憩・私的操作時に手動で切り替えた状態。
赤の丸 / アイコンなし: 異常担当者にすぐ連絡をお願いします。

こんな時はどうする？

  • 休憩中・私的利用: 一時停止。戻ったら再開

  • タスクトレイにアイコンが無い: タスクトレイの上向き矢印(^)で隠れているアイコンを表示

日次チェック（毎朝3分）

毎朝の開店準備中に、誰でもできる3項目を確認します。担当者を 1 名決めておくと運用が安定します。

朝の確認 ①
緑の丸が出ているか

  • PC起動後、タスクトレイを確認
  • 緑の丸が出ていればOK
  • 無ければ WorkScope.exe をスタートメニューから起動
  • それでもダメなら担当者に連絡

朝の確認 ②
昨日のデータがあるか

  • トレイ右クリック →「直近のスクショを確認」
  • 昨日の業務時間中の画像が残っているか目視
  • 0件なら昨日の録画が止まっていた可能性
  • 5件以上残っていれば正常

朝の確認 ③
マスキングが効いているか

  • 直近スクショから1枚開いて目視
  • 患者情報が黒塗りされているか確認
  • 少しでも怪しい部分があれば即一時停止
  • 担当者に画像を送付（マスク済みなのでOK）

WEEKLY CHECK / 06

週次チェック（毎週金曜10分）

「1週間ちゃんと回っているか」を担当者が確認します。金曜の業務終了前がおすすめです。

  1. 1
    容量チェック（2分）
    トレイ右クリック →「データフォルダを開く」→ %APPDATA%\WorkScope\data のサイズを確認。1週間で 1〜3GB が正常。0.1GB 未満なら録画停止の疑い、5GB 超なら異常容量。
  2. 2
    スクショサンプル目視（5分）
    同フォルダ内の screenshots フォルダから無作為に 10枚 開いて、すべて黒塗りが効いているか確認。怪しい画像は即削除し、ファイル名と理由をメモ。
  3. 3
    エラーログ確認（2分）
    %APPDATA%\WorkScope\logs\collector.log をメモ帳で開き、末尾の100行を確認。ERROR や CRITICAL の行があれば、行ごと担当者に共有。WARN は通常スキップで構いません。
  4. 4
    記録シートを担当者にLINE/メールで送信（1分）
    日次チェックシート＋週次の所感を担当者に送付。「マスキング異常なし」「容量◯GB」「エラー◯件」のサマリだけでOK。

異常を見つけたら（即対応フロー）

異変を見つけた瞬間に、現場で迷わず動ける手順です。

① 異変を発見 赤丸 / マスク漏れ / 動作重い / アイコン消失 etc
→
② 録画を一時停止 右クリック → 一時停止
まず止めるのが最優先
→
③ 担当者に電話 / LINE どんな小さなことでも報告。
判断は担当者がします
→
④ 指示に従う 修正版配布まで停止
もしくは即時アンインストール

パターン別の見分け方

  • 赤丸 録画エラー。即停止 → 連絡
  • マスク漏れ 個人情報漏洩リスク。即停止 → 連絡
  • PC重い 一時停止して様子見。継続なら連絡
  • アイコン消失 再起動で復活することが多い
  • 緑丸のまま 異常なし。日常運用継続

絶対にやってはいけないこと

  • 独自判断で data/ フォルダを削除する
  • 担当者連絡前に「アンインストール」を実行する（ログが残らないため）
  • 異変のスクショをメール / SNSに上げる（社外共有はNG）
  • マスク漏れがあった画像をそのまま残す（即削除）
  • 緊急時に判断を1人で抱え込む（必ず担当者に共有）

DATA HANDOVER / 09

2週間後: データの回収方法

2週間の収集完了後、暗号化USBメモリ持参で担当者がお伺いします。所要時間は約30分です。

  1. 1
    録画を一時停止（重要）
    タスクトレイ右クリック →「一時停止」→ 灰色丸を確認 → 5秒待つ。書き込み中のファイルが flush されるのを待ってからコピーします。
  2. 2
    データフォルダを暗号化USBにコピー
    %APPDATA%\WorkScope\data 配下と logs を robocopy で安全コピー。所要時間 5〜10分。失敗 0 件であることをログで確認します。
  3. 3
    薬局担当者と立会のもと、ランダム3枚を目視確認
    コピーしたデータから 3 枚をランダムに開き、マスキング状態を一緒に確認。確認シートに署名いただきます。
  4. 4
    運用継続 / 一時停止 / アンインストールを選択
    薬局のご希望で、(A) 継続録画 / (B) 一時停止 / (C) 完全アンインストール のいずれかを選択。Cの場合は uninstall.bat を管理者実行 → データも削除を選択して完全消去。

★ 持ち帰り後の取り扱い

  • 暗号化USB は社内の鍵付きキャビネットで保管します
  • データは社内の暗号化サーバへ移送後、USBは安全消去して再利用します
  • 解析作業はクラウドへ送信せず、オフラインPC上のみで実施します
  • 解析完了後の生データは、ご希望に応じて削除証明書を発行します

よくあるご質問

Q. レセコンPCの動作が重くなりませんか？
A. CPU使用率は通常2〜4%、メモリは200MB程度です。レセコン業務に支障が出る重さにはなりません。万一重さを感じたら一時停止してご連絡ください。
Q. インターネットに何か送られていませんか？
A. 一切送信しません。ファイアウォールでブロックされても正常動作します。ご不安な場合はネットワーク監視ソフトでご確認ください。
Q. アンインストールするとデータも消えますか？
A.「データも一緒に削除する／データは残す」の選択肢が出ます。回収前なら「データは残す」を選択してください。
Q. PC を再起動するとどうなりますか？
A. 自動的に再起動して録画を再開します。何もしていただく必要はありません。
Q. 業務終了後・夜間も録画されていますか？
A. 設定で「夜10時〜朝6時はキャプチャしない」も可能です。ご希望をお伝えください。
