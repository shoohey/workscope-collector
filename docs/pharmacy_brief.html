CHALLENGE / 01

薬局業務には、見えない反復作業が多い

処方箋受付・調剤・服薬指導・レセプト点検——日々の業務は多岐にわたりますが、
「どの作業に何分かかっているか」を正確に把握している薬局はほとんどありません。

  • 処方箋入力・レセコン操作などPC作業の所要時間が「肌感覚」でしか分からない
  • 本来は薬剤師でなくても可能な作業が、薬剤師の手で行われ続けている
  • 薬剤師の在席時間のうち、患者対応に使えている時間がどれだけあるか不明
  • RPAツールを入れたいが、何から自動化すべきか判断材料がない

SOLUTION / 02

WorkScope Collector で、業務を「データで」可視化

2週間の業務記録 → 業務マップ生成 → 自動化候補抽出 → RPA / AIエージェント実装。
勘や記憶ではなく、実データに基づいた業務改善のための仕組みです。

01

収集

レセコン端末で2週間、業務をそのまま記録します。

02

解析

業務マップと反復パターンを抽出し、自動化候補を抽出します。

03

RPA化

優先度の高い業務から自動化を実装し、検証します。

04

展開

本番運用へ展開し、定量的に効果を測定します。

STRENGTH / 05

強み 3 自動マスキング

薬剤師法・個人情報保護法を遵守した設計。患者個人情報は端末内で自動的に黒塗り処理され、原本は即時破棄されます。

マスキング対象

  • 患者氏名（漢字・かな・カナ）
  • 保険証番号・記号番号
  • 生年月日（複数形式に対応）
  • 電話番号・住所
  • マイナンバー・処方箋ID

処理フロー

スクショ取得 → OCR認識 → 個人情報検出 → 黒塗り処理 → マスク済み画像のみ保存 → 原本は即時破棄

マスキング辞書は薬局ごとに最適化可能。誤検出時は即日アップデートで対応します。

フェーズ2 解析（1週間）

収集データから業務マップを生成し、自動化候補に優先度スコアをつけます。

自動化候補スコアリング
