2. 弊社が取得するデータ

取得項目 具体例 取得目的

操作時刻 2026-05-28T14:32:11+09:00 各操作の所要時間・業務頻度の集計

上記すべてはJSONログとして%APPDATA%\WorkScope\data\配下に保存されます。

3. 弊社が取得しないデータ

  • 非取得画面のスクリーンショット・画像（一切取得しません。画像処理・OCR処理も行いません）
  • 非取得キーボード入力された文字の中身（パスワード・氏名・メールアドレス・金額等、入力された文字そのものは記録しません。桁数のみ記録します）
  • 非取得パスワード入力欄でのキー入力（IsPassword 属性のフィールドにフォーカスがある間は、本ツールはキー記録を完全停止します）
  • 非取得クリップボードの内容
  • 非取得マイク音声・カメラ映像
  • 非取得ファイルの中身（開いているファイルの内容を読み取ることはありません）

4. データ送信について

送信先（Google Drive 共有フォルダ）: {{ENDPOINT}}
取得したJSONログは、弊社が管理するGoogle Drive の共有ドライブにアップロードされます。
通信はすべて TLS 1.2 以上で暗号化されます。
  • 送信タイミング: 営業時間外（夜間）に自動アップロード。業務時間中の通信負荷を回避します。
  • 送信内容: 上記「2. 弊社が取得するデータ」のJSONログのみ。スクリーンショットは存在しないため送信されません。
  • 保管場所: 弊社管理の Google 共有ドライブ。お客様ごとに分離されたフォルダで管理されます。
  • アクセス権限: 弊社の本件担当者および解析担当者のみがアクセスできます。再委託は行いません。
  • 保管期間: 1年間。経過後はGoogle Drive上から自動削除されます。お客様から削除請求があった場合は、それより早く削除します。

5. お客様の権利

  1. 一時停止する権利: タスクトレイのアイコンから1クリックで操作ログの記録を一時停止できます。再開も同様に1クリックです。
  2. 同意を撤回する権利: タスクトレイから「同意を撤回」を選択することで、即座に記録停止 + ローカルデータの削除が実行されます。一度同意した後でも、いつでも撤回いただけます。
  3. データ削除を請求する権利: Google Drive上に既にアップロードされているデータについても、お客様からのご請求により速やかに削除します。
  4. アンインストールする権利: いつでも本ツールを完全にアンインストールできます。同梱の uninstall.cmd で %APPDATA%\WorkScope も含めて全消去されます。
  5. 開示請求: 弊社が保有するお客様の操作ログデータの開示請求が可能です。下記連絡先までご連絡ください。
  6. 不利益取扱の禁止: 撤回や削除請求によって、お客様または従業員の方が業務上の不利益な扱いを受けることは一切ありません。

6. 弊社の責務

  • アクセス制限: 取得したデータへのアクセスは、本件担当者と解析担当者に限定します。アクセスログは弊社内で記録・監査されます。
  • 削除応諾: お客様から削除請求を受けた場合は、原則として営業日3日以内に Google Drive 上のデータを削除し、削除完了をご報告します。
  • 再委託の禁止: 取得したデータを第三者（他社・外部委託先）に渡すこと、再委託することはありません。
  • 遠隔停止機能: 万一の事案発生時、弊社から遠隔で本ツールの動作停止およびデータ削除指示を行えるよう設計されています。
  • 漏洩時の通知: 万一データ漏洩・不正アクセス等が発生した場合は、速やかにお客様へ通知し、影響範囲・対応状況をご報告します。
  • 利用目的の限定: 取得したデータは、業務棚卸しおよび業務自動化検討の目的のみに使用します。個人の人事評価・勤務評価には一切使用しません。
v1.0 との違い（v1.1-lite の特徴）
v1.0 ではスクリーンショット取得＋OCR＋PIIマスキング処理を行っていましたが、本 v1.1-lite では画像取得を完全に廃止し、操作の構造ログ（JSON）のみを取得する仕様としました。これにより、画面に表示された個人情報・機密情報が画像として残るリスクをゼロにしています。
