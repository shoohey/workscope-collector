CONSENT FORM v1.0

業務スクリーン記録ツール 利用同意書

導入先: 大森薬局（以下「当社」または「当薬局」といいます）は、 業務の棚卸しおよび業務自動化（RPA・AIエージェント化）の検討を目的として、業務用PC上で「WorkScope Collector v1.0」（以下「本ツール」）を運用します。 本書面は、本ツールが収集する情報、収集しない情報、ご本人の権利等について明示し、ご理解のうえご署名をいただくためのものです。 本書面は本ツールのバージョン v1.0 における同意書です。v0.1.0 までで取得していなかった「フィールド情報」「キーストローク種別」を新たに記録対象に加えていますのでご注意ください。

1. 本ツールが収集する情報

  • 収集アクティブウィンドウのスクリーンショット（個人情報マスキング処理済み・端末内で黒塗り）
  • 収集ウィンドウタイトル（マスキング処理済み）
  • 収集使用アプリケーション名・操作時刻・各画面の滞在時間
  • v1.0新規フォーカス中の入力フィールド情報（「数量入力欄」「保存ボタン」のような UI 要素のラベル・識別子・階層）。値は同様にマスキングを通したうえで、PIIが含まれる場合は黒塗り済みのみ記録します。
  • v1.0新規キーストロークの種別: Tab / Enter / F1〜F12 / Ctrl・Shift・Alt の組み合わせなど操作キーの種類。文字キーは「何文字打ったか」の桁数のみで、入力された文字そのものは一切記録しません。
  • v1.0新規マウスクリックの座標と対象: クリック位置と、その近傍にあった画面要素（ボタン名等、マスキング処理済み）。

2. 本ツールが収集しない情報

  • 非収集キーボード入力された文字そのもの（patient_name や金額など、入力された文字列の中身は一切記録しません。種別と桁数のみ）
  • 非収集パスワード入力中の操作（IsPassword 属性のフィールドにフォーカスがある間は、本ツールはキーロギング自体を完全停止します。タスクトレイのアイコンが赤色に変わります）
  • 非収集クリップボードの内容
  • 非収集マイク音声・カメラ映像
  • 非収集マスキング処理前のオリジナルスクリーンショット（メモリ上で黒塗りした後、ファイルとして保存することなく即座に破棄します）

3. 個人情報の取り扱い・マスキング

  • 患者氏名・保険者番号・患者ID・生年月日・電話番号・住所・郵便番号・マイナンバー・メールアドレス等は、端末内で自動マスキング（黒塗り）されます。
  • マスキングは PaddleOCR（オフライン日本語OCR）と業界別ルール（薬局向けプロファイル）により行われ、判断不能な場合は安全側に倒して画像自体を破棄します。
  • マスキング処理前の生データは、ディスク・送信先・ログのいずれにも残しません。これは自動テストで継続的に検証されています。
  • マスキングが失敗した可能性のあるスクリーンショットは unmaskable_suspected=true として記録され、本ツールの設計により送信・保存されません。

4. データ保管・送信先

本ツールの配信方式: （USB回収）
上記が「USB回収」または空欄の場合、データは端末外に送信されません。
上記がURLの場合、暗号化通信（HTTPS/TLS）で当社の解析サーバーへアップロードされます。
  • ローカル保管: %APPDATA%\WorkScope\data\ 配下にマスク済みPNG（30日）と構造化イベント（90日）。期間経過後は自動削除されます。
  • クラウドアップロード（有効化されている場合のみ）:
    • 送信は営業時間外（夜間）に行われ、業務影響を回避します。
    • 通信はすべて TLS 1.2 以上で暗号化されます。
    • 送信先サーバーでは顧客ごとに分離された保存領域に格納し、当該顧客の管理者および当社の解析担当者のみがアクセスできます。
    • 送信は当ツールが既に黒塗り済みのデータのみを対象とし、生のスクリーンショットや入力文字は送信パスに乗りません。
  • データの取扱いは、当社の個人情報保護方針およびデータ取扱方針（同梱の data_handling_policy.html）に従います。

5. あなたの権利

  1. 一時停止する権利: タスクトレイのアイコンから1クリックで録画を一時停止できます（緑●→グレー●）。
  2. 同意を撤回する権利: タスクトレイから「同意を撤回」を選択することで、即座に記録停止 + ローカルデータの削除請求が発行されます。一度同意した後でも、いつでも書面・操作いずれの方法でも撤回できます。
  3. アンインストールする権利: いつでも本ツールを完全にアンインストールできます。同梱の uninstall.cmd で %APPDATA%\WorkScope も含め全消去されます。
  4. 開示・削除を請求する権利: ご自身に関する収集データの開示請求および削除請求が可能です。下記連絡先までご連絡ください。
  5. 不利益取扱の禁止: 撤回や請求によって、勤務上の不利益な扱いを受けることは一切ありません。

6. 利用目的

  • 業務全体の棚卸し、および反復作業の特定によるRPA・AIエージェント化の検討
  • 業務効率化に資する分析・改善提案書の作成
  • 個人の人事評価・勤務評価には一切使用しません。収集したデータは個人を特定しない形で集計され、業務単位（業務名・頻度・所要時間）で分析されます。

7. データ保管期間

端末内ローカル
　イベントログ（操作時刻・アプリ名・キー種別等）: 90日間保管後に自動削除
　マスク済みスクリーンショット: 30日間保管後に自動削除

クラウド側（有効化されている場合のみ）
　受信したアップロードデータ: 業務改善提案書納品から3ヶ月後に自動削除
　ただし、削除請求があった場合はそれより早く削除します。

8. 重要な注意事項

v1.0 で追加された記録項目について
本ツールは v1.0 で「フィールド情報」「キーストローク種別」「クリック対象」を新たに記録対象に加えました。これらは入力業務の自動化（RPA化）に必要なメタ情報を取得するためのもので、キーボード入力された文字そのものは引き続き記録しません。パスワードフィールドにフォーカスしている間は、キーロギング自体が一時停止されます。
