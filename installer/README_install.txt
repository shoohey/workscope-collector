================================================================
  WorkScope Collector  USB持ち込みインストール手順書
================================================================

このUSBには、薬局の業務フロー可視化AI「WorkScope Collector」の
インストーラ一式が入っています。

目的:
  レセコン・調剤端末の操作画面を画面キャプチャ＋OCRで自動的に
  記録し、業務フロー図を自動生成するためのデータを収集します。
  個人情報（患者氏名・処方内容など）は端末側で自動マスクされ、
  外部送信は一切行いません（オフライン動作）。

================================================================
  1. USB内のファイル一覧
================================================================

  WorkScope.exe          ・・・ 本体（Windows実行ファイル）
  install.bat            ・・・ インストール用バッチ
  uninstall.bat          ・・・ アンインストール用バッチ
  README_install.txt     ・・・ このファイル
  consent_form.html      ・・・ 同意書（薬剤師さん用、印刷推奨）
  operation_guide.html   ・・・ 操作ガイド

================================================================
  2. インストール手順（5分）
================================================================

  (1) このUSBを、データ収集対象の Windows PC に挿します。

  (2) USB内の install.bat を右クリックし、
      「管理者として実行」を選びます。
      ※ 管理者権限がなくても動作しますが、管理者で実行する方が
        確実です。

  (3) 黒いウィンドウが開き、自動でインストールが進みます。
      完了メッセージが出たら、いずれかのキーを押してください。

  (4) 同意書（consent_form.html）が
      %APPDATA%\WorkScope\docs\ にコピーされています。
      これをブラウザで開き、印刷して薬剤師さんに署名いただいて
      ください。

  (5) WorkScope が自動起動し、画面右下のタスクトレイに
      アイコンが表示されます。これで収集開始です。

  以降、PCを再起動しても WorkScope が自動起動します。

================================================================
  3. 動作確認
================================================================

  ・タスクトレイ（画面右下の^マークの中）に WorkScope アイコンが
    表示されていれば動作中です。
  ・アイコンを右クリック → 「一時停止」「再開」「終了」が選べます。
  ・収集データは %APPDATA%\WorkScope\data\ に保存されます。
    （個人情報マスク後の画像とイベントログ）

================================================================
  4. トラブルシュート
================================================================

  ◆ 「Windowsによって PC が保護されました」と表示される
     ・SmartScreen の警告です。コード署名がないため、
       初回のみこの警告が出ることがあります。
     ・「詳細情報」→「実行」で続行できます。

  ◆ ウィルス対策ソフト（ESET / Norton / McAfee 等）に
     ブロックされる
     ・PyInstaller 製の単体EXE は誤検知されやすいです。
     ・対策ソフトの「除外設定」に以下を追加してください:
         %APPDATA%\WorkScope\bin\WorkScope.exe
     ・どうしても解除できない場合は、IT担当者にこのファイルが
       社内配布の信頼済みツールであることをご連絡ください。

  ◆ install.bat をダブルクリックしても黒い画面がすぐ閉じる
     ・「管理者として実行」を試してください。
     ・それでもダメな場合は、コマンドプロンプトを開いて
         cd /d (USBドライブ):\
         install.bat
       と入力すると、エラーメッセージが画面に残ります。

  ◆ PowerShell の実行ポリシーで止まる
     ・本ツールは PowerShell スクリプトを使っていないため、
       通常はこの問題は起きません。
     ・もし起きた場合は管理者で PowerShell を開き、
         Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
       を実行してください。

  ◆ 起動はしたが、タスクトレイにアイコンが出ない
     ・タスクトレイの上向き矢印（^）をクリックして隠れている
       アイコンを確認してください。
     ・%APPDATA%\WorkScope\logs\ 配下のログを確認すると
       原因が分かることがあります。

================================================================
  5. アンインストール手順
================================================================

  (1) USB内の uninstall.bat を実行します。
      （右クリック → 管理者として実行を推奨）

  (2) 起動中のプロセスが自動停止し、スタートアップ登録も解除
      されます。

  (3) 「収集したデータも削除しますか？」と聞かれます。
      ・Y → %APPDATA%\WorkScope\ を完全削除
      ・N → プログラムだけ削除し、データはそのまま保持

  完全削除を選んだ場合、PC上に WorkScope の痕跡は残りません。

================================================================
  6. データの取り出し（USB回収時）
================================================================

  対象 PC の以下のフォルダを、USBにそのままコピーしてください:

      %APPDATA%\WorkScope\data\

  内訳:
    screenshots\  ・・・ マスク済みスクリーンショット（JPEG）
    events\       ・・・ ウィンドウ切替イベント（JSON）

  ※ ファイルが大量にある場合は ZIP圧縮を推奨します。
  ※ 個人情報は端末側で自動マスクされていますが、念のため
    USBの取り扱いには十分ご注意ください。
